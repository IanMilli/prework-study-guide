HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
<

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b 'branch-name': creates a new branch and switches to it
  • git add -A : make sure you are on the feature branch then use this to stage your changes
  • git commit -m "put message here": commits the changes to the repository
  • git pull origin main: use this code to pull in the main branch
  • git push origin feature ; use this plus the correct branch name to push changes to github
  • to then update our local main branch use use git checkout main to return to the main branch and then git pull origin main to pull the remote main branch into the local main branch

  • JavaScript

    • Note that whereas the style.css file is linked within the head element, the script.js file is linked within the body element.
    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
